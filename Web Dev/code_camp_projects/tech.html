What are advaced data types?

Advanced data types, ADTs, are specialized ways of storing data, with more mutability and customization than inbuilt data types In reality ADT is just a computer science short hand for any and all methods of storing data that don't come shipped with the programming language. However, to keep things simple, we're just going to cover 4 ADTs, each custom implmentations of commonly found data types.

These will be:

  • Linked Lists - same functionality as lists
  • Stacks and Queues - very similar to lists but with intresting constrains
  • Hashtables - a custom implmentation of dictionaries
  • Trees - these are very complex and will be explained in more detail later

Linked Lists

Linked lists are quite like traditonal lists, arrays in C, in that they're designed to sequentially store information. However, traditonal lists have many problems particulary the fact that they must be sequentially stored in memory, with results in them being relativly in efficnet and demanding.

Linked lists solve this by breaking a list down into it's most key parts. An object in memory, and the object that follows it. 

        
          class Node:
              def __init__(self, val):
                  ''' init '''
                  self._name = val
                  self._next = None

              def set_next(self, next):
                  self._next = next

              def __eq__(self, other):
                  ''' eq '''
                  if(other == None):
                      return False
                  else:
                      return self._name == other._name

              def __str__(self):
                  ''' str '''
                  return str(self._name)
